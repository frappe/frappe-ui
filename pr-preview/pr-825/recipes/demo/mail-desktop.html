Hi Sarah,

Thanks for reaching out — happy to help! That error usually happens when Slack doesn’t grant Northwind the right permissions during the connection step. Here are a few things to try:

1. Make sure you’re logged into the correct Slack workspace before starting the connection. 2. Remove Northwind from your Slack app directory, then reconnect from Settings → Integrations. 3. Confirm an admin is approving the OAuth request — restricted workspaces block it otherwise.

Let me know how it goes and I’ll dig deeper if needed.
