Focus Ring ​

An outline that marks the keyboard-focused element. It is applied globally on :focus-visible, so it shows only for keyboard navigation.

Tab through these to see it:

Retheme a ring with focus-visible:focus-ring-<color> like red for errors or green for success.

focus-ring--focus-default
focus-ring-red--focus-red
focus-ring-green--focus-green
focus-ring-amber--focus-amber
focus-ring-blue--focus-blue
focus-ring-violet--focus-violet
